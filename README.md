A responsive **Weather Application** that fetches real-time weather data using a Weather API.  
Built with **HTML, CSS, JavaScript, and Bootstrap**, this app allows users to search for any city and instantly view current weather conditions.

## ✨ Features

- 🌍 Search weather by city name  
- 🌡️ Displays temperature in °C / °F  
- 🌤️ Shows current weather conditions (Clear, Clouds, Rain, Snow, etc.)  
- 💧 Displays humidity levels  
- 🌬️ Shows wind speed  
- 🎨 Dynamic weather icons based on conditions  
- 📱 Fully responsive design  

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling  
- **JavaScript (ES6)** – Logic & API integration  
- **Bootstrap 5** – Responsive layout  
- **Weather API** – Real-time data fetching
